Class cutEllipsoid.

Método draw capaz de apagar uma elipsoide na matriz 3D.

Parâmetros
s

Método draw apaga uma elipsoide de acordo com os valores de sua localização. Para isso, é verificado se as coordenadas informadas estão dentro do raio da elipsoide para assim "excluir" os voxels dentro do espaço correspondente as suas dimensões na matriz 3D.
